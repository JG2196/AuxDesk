# AuxDesk

**Plan with purpose. Focus with intent.**

AuxDesk is a minimalist productivity app designed for forward planning and daily focus. Built with Blazor Hybrid, it provides a cleaner, distraction-free way to stay on task without juggling multiple apps, whiteboards, and sticky notes.
